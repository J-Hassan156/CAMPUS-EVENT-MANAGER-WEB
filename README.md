# CAMPUS-EVENT-MANAGER-WEB
Campus Event Manager is a full stack web application designed to manage university or campus events digitally. The system allows students to explore upcoming events, register online, and track available seats and event timings. An admin panel is included to manage events and monitor registrations efficiently.
